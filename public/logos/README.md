# Company Logos

Add the following logo files to this directory for the "Featured In" carousel:

## Required Files:

- `abc7.png` - ABC7 News logo
- `nypost.png` - New York Post logo
- `tmz.png` - TMZ logo
- `menshealth.png` - Men's Health logo
- `vanityfair.png` - Vanity Fair logo
- `cnbc.png` - CNBC logo
- `ktla.png` - KTLA logo
- `slate.png` - Slate logo
- `economictimes.png` - The Economic Times logo
- `dailymail.png` - Daily Mail logo
- `thecut.png` - The Cut logo

## Guidelines:

- Use PNG format with transparent backgrounds when possible
- Keep logos white or light colored for visibility on dark background
- Recommended height: 32-40px for optimal display
- Files should be optimized for web (compressed)

## Fallback:

If any logo file is missing, the component will automatically fall back to text display of the company name.
